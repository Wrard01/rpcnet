Programa que buasca si el archivo rpcnet se encuentra dentro de los archivos de sistema de windows
el programa busca en los directorios system32 para la version 64bit y el directorio SysWow64
para la version de 32 bits, cualquiera de estas versiones podria estar instalada en windows 64 bits.

Se recomienda presionar los dos botones para estar seguro que el archivo no se encuentrta instalado en
el ordenador, ya que una version de 32 bits podria estar instalada en windows 64 \\

rpcnet.exe es parte de Absolut Software, cons ayuda de este programa se puede verificar que los archivos
no se encuentran en la terminal que se esta trabajando, para asegurar que al ser enviada a su destino final
no tendra problemas de lojack por parte del antiguo propietarion.
